Please setup your environmet before running this code,
1. conda env update -f environment.yml (or) conda env create -f environment.yml.
2. Download the model weights from https://smpl.is.tue.mpg.de/downloads and put those weights inside ~/smplpytorch/native/models folder.
2. navigate to cloned folder and run python script by following command
   python challenge.py
3. please enter from 1-4 to see various poses as given in the PDF file.
4. you can re-run to give new poses.
